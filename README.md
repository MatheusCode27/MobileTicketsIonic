# MobileTicketsIonic

Sistema mobile de controle de atendimento desenvolvido com Ionic + Angular.

---

## Funcionalidades

- Emissão de senhas
- Fila prioritária
- Fila geral
- Fila de exames
- Painel de atendimento
- Relógio em tempo real
- Som ao chamar senha
- Interface moderna responsiva

---

## Tecnologias Utilizadas

- Ionic
- Angular
- TypeScript
- SCSS
- Capacitor

---

## Como executar o projeto

```bash
npm install
ionic serve
```
